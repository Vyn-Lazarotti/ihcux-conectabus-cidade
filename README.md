# Contexto de uso

A prefeitura de uma metrópole quer lançar o "ConectaBus", um aplicativo que unifica todo o transporte público e privado da cidade (ônibus, metrô, bicicletas compartilhadas e patinetes).

O aplicativo permitirá que usuários:

1. Planejador de Rotas: Digitar um destino e ver a combinação mais rápida de transportes.
2. Ocupação em Tempo Real: Ver se o próximo ônibus/vagão está vazio, moderado ou lotado.
3. Bilhete Único Digital: Pagar a passagem via QR Code diretamente no app.
4. Alertas de Incidentes: Receber notificações sobre linhas paradas ou greves.
5. Botão de Segurança: Reportar assédio ou perigo dentro do transporte em tempo real.
